# GitHub Global 技术文档

## 技术选型

**前端：** Next.js 16 + React 19 + TypeScript + Tailwind CSS

**后端：** Prisma ORM + MySQL 8.0 + Redis 7 + OpenAI SDK

## 系统架构

遵循 Next.js App Router 规范，单体应用结构。`src/app/api` 包含 RESTful API 路由，涵盖认证、仓库管理、翻译任务、Webhook；`src/lib` 存放核心业务逻辑，包括翻译引擎、加密工具和会话管理。

## 数据模型

User：用户信息、GitHub 凭证、API Key 及配额；Repository：仓库信息、翻译配置；TranslationTask：任务状态与进度；RateLimit：API 访问频率控制。

## 翻译引擎

基于 OpenRouter 对接多种大语言模型，通过 Prompt 模板指导翻译，完整保留 Markdown 格式。推荐 DeepSeek Chat（性价比）或 Claude 3.5 Sonnet（质量）。

## 安全与部署

敏感信息 AES-256 加密存储，Redis 会话管理，API 频率限制。部署需 Node.js 18+、MySQL 8.0+、Redis、GitHub OAuth 凭证及 OpenRouter API 权限。
